# Prerequisites
Install the following:
1. JDK 21
2. Maven

# Cloning
Clone this project and open the root directory (where `pom.xml` is).

# Build
Run the following:
`docker-compose up -d`
`.\mvnw.cmd spring-boot:run`

No to send request use a tool like *Postman*.
Use `http://localhost:8080/{api request}`.
